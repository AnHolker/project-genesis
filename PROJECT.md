# Project Genesis

Version: 0.1.0
Status: Active
Owner: Founder
Maintainer: CTO

---

# Vision

Project Genesis aims to build an AI Native Game Platform.

Instead of developing games manually, users can create, modify and interact with games entirely through natural language.

The platform should allow anyone, regardless of programming experience, to turn ideas into playable games within seconds.

---

# Mission

Build the world's best AI-powered game creation platform.

The platform should make game development as natural as having a conversation.

---

# Long-Term Goal

Project Genesis is **not** a single game.

Project Genesis is an operating system for AI-generated games.

The platform should eventually support:

- Real-time game generation
- Natural language game modification
- AI-assisted gameplay
- Persistent game worlds
- Community sharing
- Multiplayer experiences
- Extensible runtime
- AI-driven content creation

---

# Engineering Philosophy

The project follows these principles:

1. Architecture before implementation.
2. Documentation before coding.
3. AI assists development.
4. Humans make final decisions.
5. Knowledge is more valuable than code.

---

# AI Collaboration

AI is an engineer.

AI is not the architect.

AI must execute tasks within defined boundaries.

AI must never modify project direction without approval.

---

# Current Phase

Phase 0

Project Foundation

Current objective:

Build the engineering infrastructure before implementing any business functionality.

---

# Success Criteria

Project Genesis succeeds when:

- Anyone can create games using natural language.
- AI can continuously evolve games.
- Multiple AI models can collaborate on development.
- The project can be continued by any future AI without relying on chat history.
